ジーズアカデミーに入った理由

  1. プログラミングスキルを高いレベルで身につけたかったから。
  2. プログラミングをやろうやろうとしてなかなか進めない自分へのプレッシャーをかけるため。
  3. 自分は「これのプロです」と言えるスキルが欲しかったから。
  4. アプリケーションの開発を企画から設計、開発まで経験し、ビジネスマンとしてステップアップしたいと考えていたから。
  5. プログラミングを身に付けて、IoT AI 機械学習 データ分析などの分野をニュースとしてみるのではなく「当事者」としてそれらの分野に触れていたいと思ったので、そのきっかけ作り。 （いずれはデータ分析などの分野で日本のサッカー界をより高いレベルに引き上げられれば、なんて妄想をしています。）

プロダクトのゴール地点はどこに置くか

卒業プレゼン時

  1. 見た目のデザインが自分が（そして他人が見ても）納得できるものである。
  2. javascriptをある程度のレベルで理解できている状態であり、そしてその技術（できればフレームワーク）もしっかり使いこなせている。（今回の場合はNode.jsとSocket.IOでリアルタイム通信を実装する、細かいエフェクトをjQueryを使って表現する、など）。
  3. 企画がしっかり社会的に価値があり、一定数の人から評価されるものである。（もちろんここを目指すべきだが、今回ではプライオリティはそこまで高くはない）
  4. ユーザーから見て「何ができるアプリなのか」がしっかり伝わるUI / UXを作り、メッセージをしっかりと届ける。
  5. 目指せシリコンバレー！（個人的な目標）

その後

  • まずは思っている形にサービスを作り込むことをしてみる。（リアルタイムで画像に編集できる機能をつけたり、facebookログインをつけたり。）
  • とはいえ、実際に作るプロダクトの評価が低い場合はスパッと諦めてもいいかもしれない。そこは自己判断。

自分自身のゴール地点はどこに置くか

卒業プレゼン時

  1. HTML、CSS、javascriptはある程度自由に使えるようになる。
  2. PHP、MySQL、milkcocoa、Node.jsとSocket.IOを利用した開発を経験し、サーバー周りの知識も一通り身につける。
  3. 企画して作ったものをしっかりと相手に伝えるプレゼン能力をしっかりと身につける。
  4. プログラミング完全未経験から、1.2.の能力を使って新しい企画やサービスを立ち上げることができるようになる（ここが一番大事）

その後

  1. 勉強会などに積極的に参加したり、自分でアプリケーションを作ってみたりする中でNode.jsやmilkcocoaでの開発を経験する。（1~2年の長期スパンで）
  2. 社内にエンジニアがいないので、社内に「エンジニア」というポジションを作る
  3. 条件の良さそうなところがあれば転職も考える（ただし、これはファーストプライオリティではない）
  4. 初心者にプログラミングを教える活動をする（ライフワークとして）

身に付けたい言語・技術

  • HTML / CSS
  • PHP MySQL
  • Javascript
  • Node.js Socket.IO
  • Milkcocoa

卒業後に深めていきたい部分

  • Javascript
  • Node.js Socket.IO
  • Milkcocoa

今後そのスキルを使って何をしたいか？

  • プログラミング教育の分野で経験を積む
  • 自分のやってみたいアプリの開発（今回のように、数ヶ月に1個ペースで。）
  • IoTやAIの分野、機械学習への学習チャレンジ（ざっくりしているが、ここが一番興味があるところです。）
